# Loginoa Studio Website

GitHubの `homepage` リポジトリ直下へ、ZIPではなくこのフォルダ内のファイルをアップロードします。

## ルーレット＋の公開時に変更する場所

現在はトップページと `roulette-plus/index.html` で「App Store 近日公開」と表示しています。
App Store公開後は、次の2ファイルにApp Storeリンクを追加してください。

- `index.html`
- `roulette-plus/index.html`

プライバシーポリシーのURLは `roulette-plus-privacy/` です。
